Image Search Abstraction Layer

This simple app is made for a FreeCodeCamp code challenge: Link to FCC

Examples:

https://image-search-microserver.herokuapp.com/api/imagesearch/searchkeyword
https://image-search-microserver.herokuapp.com/api/imagesearch/searchkeyword?offset=3
https://image-search-microserver.herokuapp.com/api/history

Other info:

  • History is limited to 10 records. Older ones get replaced
  • Each page (offset) returns 10 results.
  • You are using my private Google Api key which is limited to 100 request per day, please don't exploit it.
