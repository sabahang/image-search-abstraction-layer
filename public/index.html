The Timestamp Microserver

This simple app is made for a FreeCodeCamp code challenge: Link to FCC

Server is supposed to parse the Date string you send it as input and return Unix timestamp and Human readable time format, if the parameter is not pars-able, you'll get null

Examples:

https://timestamp-microserver-sabahang.c9users.io/December%2015,%202015
https://timestamp-microserver-sabahang.c9users.io/1450137600

Examples Output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
